ASCII Art from Image telegram bot
=================================

Asynchronous telegram bot technologies:

*   Python 3.8.5
*   Aiogram 2.9.2
*   ImgKit 1.0.2
*   Ascii magic 1.5.4

* * *
Telegram bot which converts the user's picture into a set of chars. There are two types of ouput picture - HTML, ASCII.
